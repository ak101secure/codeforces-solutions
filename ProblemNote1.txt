Topics:
üß† Mo's Algorithm (Query Square Root Decomposition)
- Offline algorithm to answer range queries in O((N + Q)‚àöN)
- Ideal for static queries like ‚Äúhow many distinct numbers in a range?‚Äù

üå≤ Heavy Light Decomposition (HLD)
- For path queries in trees
- Used with segment trees to answer path-based queries efficiently

üß© Centroid Decomposition
- Used in divide & conquer on trees
- Efficient for solving path problems in trees

üåÄ Treaps / Splay Trees / Ordered Sets
- Balanced BSTs with fast insert/delete/find
- STL alternatives: `std::set`, `pbds` in C++

üåä Max Flow / Min Cut (Ford-Fulkerson, Dinic's)
- Network flow problems
- Matching, assignment, circulation problems
